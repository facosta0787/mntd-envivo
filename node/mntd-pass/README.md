# Usage
<!-- usage -->
```sh-session
$ npm install -g mntd-pass
$ mntd-pass COMMAND
running command...
$ mntd-pass (-v|--version|version)
mntd-pass/1.0.0 linux-x64 node-v12.16.1
$ mntd-pass --help [COMMAND]
USAGE
  $ mntd-pass COMMAND
...
```
